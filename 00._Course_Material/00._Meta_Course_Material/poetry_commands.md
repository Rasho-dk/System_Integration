# poetry

Creates a toml file according to the PEP 518 standard!

```bash
$ poetry init
```

With non-interactive mode:

```bash
$ poetry init -n
```

```bash
$ poetry install
```

```bash
$ poetry shell
```

To install additional libraries:

```bash
$ poetry add <library>
```

To initialize a project with a structure that includes tests etc.

```bash
$ poetry new <project_name>
```
